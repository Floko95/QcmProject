Afin de faire fonctionner le projet il est nécessaire d'implementer dans sa base de donnée le schéma relationnel du projet, et ce à l'aide des trois scripts sql inclus dans le projet.

Si l'utilisateur souhaite mettre en place ce schéma relationnel dans sa base de données, il faut utiliser les 2 scripts suivant dans cet ordre:

1-init_BDD.sql
2-insert_BDD.sql

Si l'utilisateur souhaite supprimer ce schéma, il faut utiliser le script reset_BDD.sql.
